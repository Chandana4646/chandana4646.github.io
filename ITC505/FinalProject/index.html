Lights Out!

Instructions:
Click on a square to toggle its light and the lights of its neighbors. you need to turn off all the lights to win.

Adendum
This document describes the development process, creative choices, and challenges encountered during the creation of the Lights Out game. It also highlights the insights gained along the way.

Development Process:
Initial Setup: The first step involved structuring the HTML and CSS files for the game layout and styling.
JavaScript Logic: The core functionality was built through JavaScript functions for:
Board creation and square initialization.
Square toggle on click and adjacent squares flip.
Win condition check and click count display.
Random board generation with backtracking to ensure solvability.

Testing and Debugging: The game logic was thoroughly tested and debugged to ensure smooth gameplay and accurate win detection.

Creative Choices:
Visuals: A simple and clean design was chosen to focus on gameplay and avoid distractions.
Click Count Display: Adding a click counter adds an additional element of challenge and accomplishment.
Win Message Customization: The win alert message was personalized to include the number of clicks for a more satisfying experience.

Challenges and Solutions: Win Condition Optimization: Initially, the win condition checked for all squares to be off, which could be inefficient. Optimizing it to check for only one remaining light greatly improved responsiveness.
Insights Gained:
JavaScript DOM Manipulation: The development process deepened understanding of manipulating DOM elements through JavaScript, including creating, styling, and adding event listeners.
Game Design Considerations: The project provided valuable insights into game design principles, such as balancing difficulty, providing clear instructions, and incorporating feedback mechanisms.

Overall, the development process of the Lights Out game was a valuable learning experience, allowing for exploration of new concepts and refining existing skills. The final product serves as a fun and challenging game that demonstrates the power of JavaScript and game design principles.
